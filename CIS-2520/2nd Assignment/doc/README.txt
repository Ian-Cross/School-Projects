*******Compiling the programs*******

To compile the car rental program execute the following command from the root directory:
gcc src/q1.c src/fileOpening.c src/fileParsing.c src/linkedList.c src/menu.c -o bin/q1

or type:

make q1

To compile the calculation program execute the following command from the root directory:
gcc q2.c -o bin/q2

or type:

make q2

*******Using The program*******

To run the car rental program type the following command from the root directory of the assignment
./bin/q1

or type:

make rq1

Once the program has been executed a menu will come up and prompt the user for input. The diskfile will
be automatically opened and parsed, loading all the cars into their lists
Everything from that point on everything is handled by the program

To run the car rental program type the following command from the root directory of the assignment
./bin/q2 *equation here*

or type:

make rq2

Once the program has been executed the calculation will be made and the answer will be displayed on screen
